Getting Started

CodeRandom is a game designed to teach very basic cryptography skills. In this game you are given an encoded word you have to decipher. To decipher the certain message, you need to figure out each rule that may have been used, and decode the master word in order of each card/hint to win!
If you are still confused, I'll walk you through an example puzzle.
